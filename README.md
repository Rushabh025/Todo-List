# Todo list app
A simple Todo list with vanilla JS 

Demo Link : https://rushabh025.github.io/Todo-List/


### General steps to follow when creating a project

- Thinking about the UI
- Functionality
	- Add a TODO
	- Delete a TODO
	- check task
	- total items count
- Data
	- tasks - an array
	- task - {done, text, id}
- Functions (in code)
	- addTodo
	- deleteTodo
	- checkTodo
	- renderTodosList
	- showNotification
